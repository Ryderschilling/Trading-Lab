## Your code is already on GitHub!

## If you made changes and want to update GitHub:

### Step 1: Open Terminal
Open Terminal app on your Mac

### Step 2: Go to your project folder
```bash
cd "/Users/ryderschilling/AI/Robinhood app"
```

### Step 3: Add all changes
```bash
git add .
```

### Step 4: Save changes
```bash
git commit -m "Update project"
```

### Step 5: Push to GitHub
```bash
git push
```

**Done!** Your changes are now on GitHub.

---

## If you need to set up GitHub for the first time:

1. Go to: https://github.com/new
2. Name it: `Trading-Lab`
3. Click "Create repository"
4. Then in Terminal, run:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/Trading-Lab.git
   git push -u origin main
   ```
